安装 / 启动 Docker

本工具全程依赖 Docker，需常驻运行。以下是各平台的安装步骤。

Apple 芯片(M1/M2/M3/M4)下，EasyConnect / aTrust 镜像走原生 arm64，无需 Rosetta。Intel Mac 同样支持。
  1. 判断芯片，下载对应安装包

    点击左上角苹果菜单 → 关于本机，查看「芯片」栏。

    • 显示 Apple M… → 下载 Apple Silicon 版
    • 显示 Intel → 下载 Intel Chip 版

    下载地址：docker.com/products/docker-desktop

  2. 安装并启动

    打开下载的 .dmg，把 Docker.app 拖入「应用程序」文件夹，然后双击启动。等待菜单栏右上角的鲸鱼图标不再闪烁、变为稳定状态，即表示 Docker 已就绪。

  3. 终端验证

    打开「终端」，运行：

    docker version

    正常输出包含 Client 和 Server 两段版本信息，说明 Docker Daemon 已正常连接。

  4. 保持 Docker 常驻

    本工具运行时需要 Docker 一直开着。建议在 Docker Desktop → Settings → General 中勾选 Start Docker Desktop when you sign in to your computer，避免每次手动启动。

Windows 版 Docker Desktop 依赖 WSL 2 作为后端，需先完成 WSL 2 初始化。
  1. 启用 WSL 2

    以管理员身份打开 PowerShell，运行： 

    wsl --install

    命令会自动启用虚拟化功能并安装 Ubuntu 子系统。安装完成后重启电脑。

  2. 下载并安装 Docker Desktop for Windows

    前往 docker.com/products/docker-desktop，下载 Windows 安装包（Docker Desktop Installer.exe）。运行安装向导，保持默认选项（Use WSL 2 based engine）即可。

  3. 启动并等待就绪

    安装完成后启动 Docker Desktop。等待任务栏托盘区鲸鱼图标稳定，点击图标可看到状态显示 Docker Desktop is running。

  4. 终端验证

    打开 PowerShell 或 Windows 终端，运行：

    docker version

    正常输出包含 Client 和 Server 两段版本信息，说明 Docker Daemon 已正常连接。
