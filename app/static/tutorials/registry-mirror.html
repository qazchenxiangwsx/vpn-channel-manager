配置 Docker 国内镜像源

配好后所有 docker pull 走国内源，不必每次单独拉。

镜像源会变动或失效，若某个不通请换另一个。以下源与「环境诊断 → 镜像源」屏里维护的保持一致，当前实测可用（2026-06）：docker.1ms.run、hub.rat.dev。
  1. 打开 Docker Desktop 设置

    点击菜单栏右上角鲸鱼图标，选择 Settings（或点击 ⚙️）。

  2. 进入 Docker Engine 配置

    在左侧导航中点击 Docker Engine，右侧会出现一个可编辑的 JSON 配置框。

  3. 加入 registry-mirrors

    在 JSON 中加入以下字段。若已有其他键，合并进去，注意保持逗号正确： 

    {
  "registry-mirrors": [
    "https://docker.1ms.run",
    "https://hub.rat.dev"
  ]
}
  4. Apply & Restart

    点击右下角 Apply & Restart，等待 Docker 重启完成（菜单栏鲸鱼图标稳定后即就绪）。
